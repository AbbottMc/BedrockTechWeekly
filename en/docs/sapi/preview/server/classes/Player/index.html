Class: Player

Represents a player within the world.

Hierarchy​

  • Entity

    ↳ Player

Properties​

dimension​

• Readonly dimension: Dimension

Dimension that the entity is currently within.

Throws

This property can throw when used.

Overrides​

Entity.dimension

id​

• Readonly id: string

Unique identifier of the player. This identifier is intended to be consistent across loads of a world instance. No meaning should be inferred from the value and structure of this unique identifier - do not parse or interpret it.

Throws

This property can throw when used.

Overrides​

Entity.id

isSneaking​

• isSneaking: boolean

True if the player is currently using a sneaking movement.

Overrides​

Entity.isSneaking

level​

• Readonly level: number

location​

• Readonly location: Vector3

Current location of the player.

Throws

This property can throw when used.

Overrides​

Entity.location

name​

• Readonly name: string

Name of the player.

Throws

This property can throw when used.

nameTag​

• nameTag: string

Optional name tag of the player.

Overrides​

Entity.nameTag

onScreenDisplay​

• Readonly onScreenDisplay: ScreenDisplay

Contains methods for manipulating the on-screen display of a Player.

scoreboard​

• Readonly scoreboard: ScoreboardIdentity

Returns a scoreboard identity that represents this entity.

Throws

This property can throw when used.

Overrides​

Entity.scoreboard

selectedSlot​

• selectedSlot: number

Manages the selected slot in the player's hotbar.

spawnDimension​

• Optional Readonly spawnDimension: Dimension

target​

• Readonly target: Entity

Retrieves or sets an entity that is used as the target of AI-related behaviors, like attacking. For players, which don't use any AI semantics, this property does not do anything.

Throws

This property can throw when used.

Overrides​

Entity.target
