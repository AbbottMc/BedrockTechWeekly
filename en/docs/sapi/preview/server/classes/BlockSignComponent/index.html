Class: BlockSignComponent

Represents a block that can display text on it.

Hierarchy​

  • BlockComponent

    ↳ BlockSignComponent

Properties​

block​

• Readonly block: Block

Remarks

Block instance that this component pertains to.

Inherited from​

BlockComponent.block

typeId​

• Readonly typeId: string

Inherited from​

BlockComponent.typeId

componentId​

▪ Static Readonly componentId: "minecraft:sign"

Remarks

Identifier of this component. Should always be minecraft:sign.

Methods​

getRawText​

▸ getRawText(): RawText

Remarks

Returns the RawText of the sign if setText was called with a RawMessage or a RawText object

Throws

This function can throw errors.

Returns​

RawText

getText​

▸ getText(): string

Remarks

Returns the text of the sign if setText was called with a string

Throws

This function can throw errors.

Returns​

string

getTextDyeColor​

▸ getTextDyeColor(): DyeColor

Remarks

Gets the dye that is on the text or undefined if the sign has not been dyed

Throws

This function can throw errors.

Returns​

DyeColor

setText​

▸ setText(message): void

Remarks

Sets the text of the sign component

This function can't be called in read-only mode.

Throws

This function can throw errors.

Example

SetRawMessage.ts

       const helloWorldMessage: RawMessage = { text: 'Hello World' };
       sign.setText(helloWorldMessage);

       // Sign text will be saved as a RawText
       const result: RawText = sign.getRawText();
       JSON.stringify(result); // { rawtext: [{ text: 'Hello World' }] };

Example

SetRawText.ts

       const helloWorldText: RawText = { rawtext: [{ text: 'Hello World' }] };
       sign.setText(helloWorldText);

       // There will be no data transformation unlike calling setText with a RawMessage
       const result: RawText = sign.getRawText();
       JSON.stringify(result); // { rawtext: [{ text: 'Hello World' }] };

Example

SetString.ts

       // Set sign to say 'Hello'
       sign.setText('Hello');
       sign.getText(); // 'Hello'

Parameters​

NameTypeDescription
messagestring | RawText | RawMessageThe message to set on the sign. If set to a string, then call getText to read that string. If set to a RawMessage, then calling getRawText will return a RawText. If set to a RawText, then calling getRawText will return the same object that was passed in.

Returns​

void

setTextDyeColor​

▸ setTextDyeColor(color?): void

Remarks

Sets the dye color of the text

This function can't be called in read-only mode.

Throws

This function can throw errors.

Parameters​

NameTypeDescription
color?DyeColorThe dye color you want or undefined if you want to clear the dye on the sign

Returns​

void
